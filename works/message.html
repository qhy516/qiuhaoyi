分布式消息系统

项目概述

本项目是一个基于 RabbitMQ 的分布式消息处理平台，实现了可靠的消息投递、延迟消息处理、死信队列处理等核心功能。系统采用集群部署方式，支持消息的持久化存储，并实现了完整的消息监控和管理功能。

系统架构

整体架构

  • 分布式消息队列集群
  • 高可用集群架构
  • 消息持久化机制
  • 消息监控管理平台

技术架构

  • 消息中间件：RabbitMQ Cluster
  • 开发框架：Spring Boot + Spring Cloud Stream
  • 持久化存储：MySQL + MongoDB
  • 缓存系统：Redis Cluster
  • 监控系统：RabbitMQ Management
  • 链路追踪：SkyWalking
  • 日志系统：ELK Stack
  • 告警系统：Prometheus + Grafana
  • 容器化部署：Docker + K8s
  • 负载均衡：HAProxy
